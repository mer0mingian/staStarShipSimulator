# Starship Weaponry Reference

## Weapon Categories

All starship weapons have:
- **Type:** Energy or Torpedo
- **Range:** Close, Medium, or Long (cannot attack beyond listed range)
- **Damage:** Base damage + Weapons Rating bonus
- **Qualities:** Special rules/effects

### Damage Bonus from Weapons Rating

| Weapons Rating | Damage Bonus |
|----------------|--------------|
| 6 or lower | +0 |
| 7-8 | +1 |
| 9-10 | +2 |
| 11-12 | +3 |
| 13+ | +4 |

## Weapon Qualities

| Quality | Effect |
|---------|--------|
| **Area** | Additional targets in same zone cost 1 Momentum each (Repeatable). May Succeed at Cost |
| **Calibration** | Requires Prepare minor action before Attack major action in same turn |
| **Cumbersome** | Attack Difficulty +1. Torpedoes cannot be fired in salvo |
| **Dampening** | Target loses Reserve Power if available |
| **Depleting** | Cannot cause shaken, but Momentum cost to increase damage reduced to 1 |
| **Devastating** | Repair tasks for breaches from this weapon: Difficulty +1 |
| **Hidden X** | Concealed from scans unless specific scan attempted (Difficulty X) |
| **High Yield** | If attack causes breach(es), inflict 1 additional breach or +1 Potency |
| **Intense** | Increase damage by spending 1 Momentum instead of 2 (Repeatable) |
| **Jamming** | Until end of round: target's Communications/Sensors tasks: Difficulty +1 |
| **Persistent** | Spend 1-3 Momentum. Each round, target suffers half weapon damage (round up) for that many rounds |
| **Piercing** | Ignores target's Resistance |
| **Slowing** | Until end of round: target cannot use Keep the Initiative Momentum spend |
| **Spread** | Devastating Attack Momentum spend reduced to 1 (Repeatable) |
| **Versatile X** | Successful attack gains X bonus Momentum (cannot be saved) |

## Torpedoes

**Attack Difficulty:** 3  
**Threat Cost:** +1 immediately when attacking

### Salvos

**Cost:** +3 Threat (instead of +1)  
**Benefits:**
- Damage +1
- Gains either Area or Spread quality (attacker chooses before attack)

## Tractor Beams

**Strength:** Ship's Scale - 1  
**Activation:** Difficulty 2 Control + Security, assisted by Structure + Security  
**Effect:** Target immobilized, cannot move until breaking free

### Breaking Free Options

Target must attempt task with Difficulty = tractor beam's Strength:

| Method | Task |
|--------|------|
| **Disable Emitter** | Control + Security, assisted by Weapons + Security |
| **Disrupt Emitter** | Reason + Science, assisted by Sensors + Science |
| **Modulate Shields** | Insight + Engineering, assisted by Structure + Security |
| **Raw Power** | Daring + Conn, assisted by Engines + Conn |

## Improvised Armaments

Any weapon type can be improvised as:
- Modified phaser emitters
- Customized torpedo warhead
- Energy/particle stream from main deflector
- Create as trait during scene

## Quick Reference

**Energy Weapons:**
- Difficulty 2
- Various types with different qualities
- Delivery system determines range and base damage

**Torpedoes:**
- Difficulty 3
- +1 Threat per attack (+3 for salvo)
- Generally more powerful but harder to use
- Most require Calibration

**All Weapons:**
- Add Weapons Rating bonus to damage
- Range limits apply strictly
- Qualities modify attack effects
